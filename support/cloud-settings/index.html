Cloud Settings

Compare and Deploy your site to the cloud. 

YouDoCMS integrates with the five major cloud providers AWS, Cloudflare, Google Firebase, Github and Microsoft Azure. YouDoCMS can be used to compare the functionality and speed of each cloud allowing you to choose the the most suitable cloud for your site. The free version of YouDoCMS includes default deployment to Github pages. 

Our static site generation process builds the site using either localhost, the selected cloud CDN endpoint or a custom domain. Each cloud can be assigned a different domain prefix ie www2, www3. Therefore a site can be replicated across multiple clouds for comparison. Similarly this method could be used to generate different versions of a site if you are implementing a new look and feel.
