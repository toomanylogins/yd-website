Firebase Project

Google Storage and Firebase services are ideal for hosting your static site. Google Cloud offers two ways to serve your static site. Similar to AWS you can use storage buckets to serve your site or use the Firebase content delivery network. YouDoCMS integrates with the Firebase command line application to initialise your project and deploy your files to a custom domain. Unlike AWS Google does not provide a DNS solution so you will need to edit your DNS records to set up your domain. However the provision of SSL and storage is integrated as a single step when you initialise and create a project.

  • Single click install Firebase CLI
  • Project Initialisation and deletion.
  • Firebase SSL provisioning.
  • Optional Google storage buckets support.

Firebase Hosting

Google Firebase Hosting provides an integrated repository including storage and is the simplest way to serve your static site via Google CDN. 

  • Custom domain and SSL setup.
  • Firebase object storage with rollback.
  • Single click deploy of single page or whole site.
  • Local domain for testing.
