Joomla and Wordpress Import

YouDoCMS supports direct conversion of Joomla! 3 or WordPress 5 sites into a static equivalent. On import all existing content is imported as html and the relationships between articles, categories, pages, and posts match the originals. URLs are updated with the static file paths using the Joomla! SEF or WordPress permalink settings.

  • No hosted server. Your computer your data.
  • Multiple sites.
  • Backup included.
  • 100% secure cannot be hacked.
More Details
 
 

Mysql Database Import

Our desktop application includes a MySQL/ Maria connector and importing your data is as simple as entering the IP address, username and password. We are able to detect the table prefix and import all of the default Joomla! or Wordpress data at the click of a button.

  • Single click import from Mysql or via sql dump.
  • Joomla SEF settings preserved inc html suffix.
  • Wordpress permalinks settings imported.
  • All static urls match the CMS originals.
More Details

FTP Import images and source files. Create Reverse Proxy.

Connect to your Joomla! or WordPress site and download all your existing images and videos. If you are going to use your existing CMS template use the FTP browser to select the template css and js folders and download to your local computer.

  • All file paths to css, js, images css and vidoes are maintained.
  • Select multiple source folders.
  • Combine your static site with your CMS with a reverse proxy.
  • Setup your cloud to route any URL to your existing backend.

How to import your website.

So here are three working steps to create a JAMstack version of your site.

1

Connect to MySQL

Enter your database name, user name and password for the Joomla or WordPress site. You can find this in global configuration database settings.

2

Download FTP images and style sheets

Download your existing images and CSS files from your current site. 

3

Multiple sites

YouDoCMS can use any html template. To get started we recommend creating a JAMstack version using your current CMS theme.

How It Works?

All menu items are replicated within our database. We preserve the hierarchical menu structure and the positions of each menu on the page. 

Menus retain their status published, unpublished, trashed etc. and each menu can be reorganised using drag-and-drop. Joomla! users will be familiar with the menu manager which also displays the modules available for each menu.

Frequent users will appreciate the speed a native application brings to the editing process enabling users to concentrate on creating content.
