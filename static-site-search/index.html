Static Site Search

  1. Home
  2. Static Site Search

Static site search Is more complex than the content management system equivalent where text is indexed in a database. For static site search the index needs to be created when the pages are built. This index is then used to search the static pages. YouDoCMS integrates with the static search provided by Algolia and each time the site is built a new index is created and uploaded to Algolia by their rest API.
